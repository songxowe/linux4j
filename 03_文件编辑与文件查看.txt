1,vi 是一个在文本界面下运行的一个编辑器
	打开一个文件
		vi 普通文件路径
	vi有3种工作模式
		命令模式:光标的移动，文本的删除，复制，粘贴，查找
		插入模式:光标的移动，文本的输入，删除
		底行模式:文件的保存与退出，设置vi的操作参数x
		同一时刻vi只能处于一种模式

		模式转换
		命令模式->插入模式  i ,o,a
		插入模式->命令模式  esc

		命令模式->底行模式 :
		底行模式->命令模式 esc 或者底行模式命令执行成功

		底行模式下的命令:
			set [no]nu  行号开关
			syntax off|on 语法高亮
			set tabstop = 4 tab键的空格数
			set [no]hlsearch 搜索高亮开关
			!command 执行linux命令
			w 把修改的内容写入文件
			q 退出vi
			wq 把修改的内容写入文件,并退出vi
			q! 不保存退出vi
			w 文件名 文件另存为
			file 查看当前编辑文件的信息
		永久性设置vi的操作参数
			在用户的家目录下创建一个名字叫做.vimrc的文件
			set nu
			syntax on

			当运行vi时，首先会读取该文件的内容对vi的操作参数进行设置

		插入模式下的命令:
			x 删除当前光标所在的一个字符
			X删除当前光标前的一个字符

			dd删除当前行
			u撤销上一个操作

			使用alt + f1-6 可以打开6个控制台窗口
			2dd 从当前行开始删除2行

			yy复制当前行到粘贴板
			p从粘贴板把内容粘贴到当前光标的位置
			2yy 

			/  向上搜索字符串
			?  向下搜索字符串
			N 继续往上搜索
			n  继续往下搜索

			0 把光标移动到当前行的最前
			$ 把光标移动到当前行的最后

			H 移动到当前屏的第一行
			L  移动到当前屏的末行

			G 把光标移动到文件的最后一行
			gg把光标移动到文件的第一行
			23G把光标移动到文件的第23行
	
2,查看文件内容命令(程序,文件)
			cat 
			tac
			nl  加行号查看文件的内容
				shift + pageup/pagedown 实现翻屏

				N
